### Binary Search Tree Project

In this project, I build a balanced binary search tree.

This project is part of The Odin Project curriculum: https://www.theodinproject.com/lessons/ruby-binary-search-trees

A complete version of the project is contained in main.rb.

I wrote the educational pseudocode.rb file to solidify my understanding of the delete algorithm, which I found hard to understand and implement.

After finishing the project including the driver script (driver_script.rb), I noticed that not all methods were tested by the script, so I added another driver script in a separate file named gpt_driver.rb (generated by ChatGPT and modified by me).

Again, the ChatGPT code was tacked on in a separate file in addition to the already-completed project, and I relied on it only to expediate the testing process AFTER I had completely fullfilled the assignment instructions.
